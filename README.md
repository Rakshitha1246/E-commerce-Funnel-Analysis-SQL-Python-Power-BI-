# 🛒 E-Commerce Funnel Analysis

## 📌 Project Overview
An interactive Power BI dashboard analyzing 55,000+ user journeys across funnel stages — built to identify where users drop off and how to improve conversion rates for e-commerce businesses.

---

## 🎯 Business Problem
E-commerce businesses lose potential revenue when users browse but never buy. This project pinpoints **exactly where and why users drop off** and recommends data-backed fixes.

---

## 🔍 Key Insights
- 👀 **54,790 users** viewed products but only **1,480 added to cart** (~2.7%)
- 🛒 Cart-to-purchase conversion is strong at **~33%**
- 📉 Overall conversion rate is only **0.9%** — the product page is the bottleneck
- ⚠️ **~97% of users drop off** before adding to cart

---

## 💡 Business Recommendations
- Improve **product page design** to reduce early drop-off
- Add **urgency triggers** (limited stock, offers) to encourage cart additions
- Optimize **mobile experience** where most drop-offs likely occur
- A/B test **CTAs and product images** to improve engagement
