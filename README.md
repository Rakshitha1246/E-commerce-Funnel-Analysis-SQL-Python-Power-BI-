# 📊 E-commerce Funnel Analysis

## 🔹 Project Overview
This project analyzes user behavior in an e-commerce platform to understand how users move through the funnel stages: **View → Add to Cart → Purchase**.

---

## 🔹 Objectives
- Identify user drop-offs at each stage  
- Calculate conversion rates  
- Provide business insights to improve performance  

---

## 🔹 Tools & Technologies
- **Python** – Data cleaning & preprocessing  
- **SQL (MySQL)** – Data analysis & querying  
- **Power BI** – Dashboard & visualization  

---

## 🔹 Key Insights
- ~2.7% users move from view to cart (high drop-off)  
- ~97% users leave before adding to cart  
- Cart to purchase conversion is strong (~33%)  
- Overall conversion rate is low (~0.9%)  

---

## 🔹 Business Recommendations
- Improve product pages to reduce drop-off  
- Optimize offers to increase conversions  

---

## 🔹 Dashboard
The dashboard includes:
- KPI cards (Total Users, Purchases, Conversion Rate)  
- Funnel visualization (View → Cart → Purchase)  
- Users by stage chart  
- Key insights and business recommendations  

---

## 🔹 Conclusion
The analysis highlights a major drop-off at the initial stage, indicating the need for improved user engagement strategies to enhance overall conversion.
